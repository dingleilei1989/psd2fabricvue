# PSD 海报编辑器

一个基于 Vue 3 + ag-psd + Fabric.js 的在线海报编辑器，支持导入 PSD 文件并进行文字编辑。

## 功能特性

- 🎨 **PSD 文件导入**: 使用 ag-psd 库解析 PSD 文件
- ✏️ **文字编辑**: 添加和编辑文字，支持字体、大小、颜色设置
- 🖼️ **图层管理**: 显示和操作 PSD 图层
- 🔧 **对象操作**: 删除、层级调整等功能
- 💾 **导出功能**: 将编辑后的海报导出为 PNG 图片

## 技术栈

- **Vue 3**: 前端框架
- **Vite**: 构建工具
- **ag-psd**: PSD 文件解析库
- **Fabric.js**: Canvas 操作库

## 安装和运行

1. 安装依赖:
```bash
npm install
```

2. 启动开发服务器:
```bash
npm run dev
```

3. 访问 http://localhost:3000

## 使用说明

### 1. 导入 PSD 文件
- 点击左侧工具栏的"导入PSD文件"按钮
- 选择本地的 .psd 文件
- 系统会自动解析文件并显示图层列表

### 2. 文字编辑
- 在"文字工具"区域输入文字内容
- 调整字体大小、颜色和字体家族
- 点击"添加文字"按钮将文字添加到画布
- 可以直接在画布上拖拽和调整文字位置

### 3. 图层操作
- 解析 PSD 后，图层会显示在"PSD图层"区域
- 点击"添加到画布"可以将单个图层添加到编辑画布

### 4. 对象管理
- 选中画布上的对象后可以进行删除、层级调整
- 支持置于顶层、置于底层等操作

### 5. 导出作品
- 点击"导出图片"按钮将当前画布内容保存为 PNG 文件

## 开发说明

### 组件结构
- 左侧工具栏包含文件操作、文字工具、对象操作等功能
- 右侧为画布区域，使用 Fabric.js 进行图形操作

### 关键功能实现
- PSD 解析使用 `ag-psd` 的 `readPsd` 函数
- 画布操作基于 `fabric.Canvas`
- 文字添加使用 `fabric.Text` 对象
- 图层处理支持递归解析嵌套图层
