# 使用说明

## 项目已创建完成！

这是一个基于Vue 3的PSD转Fabric.js工具，具有以下特性：

### 🎯 核心功能
1. **PSD文件上传解析** - 支持拖拽上传PSD文件
2. **文字图层保留** - 文字图层转换为可编辑的Fabric.js文本对象
3. **图层树显示** - 可视化图层结构，支持显示/隐藏控制
4. **实时预览** - Fabric.js画布实时预览转换结果
5. **多格式导出** - 支持PNG、JPEG、SVG、JSON格式导出

### 🚀 快速开始

1. **访问应用**
   ```
   http://localhost:3000
   ```

2. **尝试演示数据**
   - 点击左侧"加载简单演示"或"加载复杂演示"按钮
   - 查看图层树和画布预览效果
   - 尝试切换图层显示/隐藏

3. **上传PSD文件**
   - 拖拽PSD文件到上传区域
   - 等待解析完成
   - 点击"转换为Fabric.js"按钮

### 📁 项目结构

```
src/
├── components/
│   ├── PsdUploader.vue      # PSD上传组件
│   ├── LayerTree.vue        # 图层树组件
│   └── FabricCanvas.vue     # Fabric.js画布组件
├── utils/
│   ├── psdToFabricConverter.js  # PSD转换器
│   └── demoData.js              # 演示数据
```

### 🔧 技术栈

- **Vue 3** - 前端框架
- **Element Plus** - UI组件库
- **Fabric.js** - 画布库
- **ag-psd** - PSD解析库
- **Vite** - 构建工具

### 💡 主要特性说明

#### 文字图层处理
- 自动识别PSD中的文字图层
- 保留文字内容、字体、大小、颜色等属性
- 在Fabric.js中创建为可编辑的Text对象
- 如果文字解析失败，会自动回退到图片模式

#### 图层控制
- 图层树显示所有图层结构
- 支持分组图层的展开/折叠
- 复选框控制图层显示/隐藏
- 点击图层名称可在画布中选中

#### 画布操作
- 鼠标滚轮缩放
- 拖拽平移
- 重置视图
- 适应屏幕大小

#### 导出功能
- PNG/JPEG图片导出
- SVG矢量图导出
- Fabric.js JSON数据导出
- 支持自定义导出参数

### 🎨 演示数据说明

项目包含两套演示数据：

1. **简单演示** - 包含基本的矩形、圆形、文字图层
2. **复杂演示** - 包含分组图层、多种文字样式、复杂布局

### 📝 开发说明

#### 添加新的图层类型
1. 在`psdToFabricConverter.js`中扩展`getLayerType`方法
2. 添加对应的`create[Type]Object`方法
3. 在图层树组件中添加相应的图标和样式

#### 自定义文字处理
- 修改`extractTextInfo`方法来支持更多文字属性
- 扩展Fabric.js文字对象的属性映射

#### 扩展导出格式
- 在`FabricCanvas.vue`中的`confirmExport`方法中添加新格式
- 实现相应的转换和下载逻辑

### 🐛 已知限制

1. PSD复杂效果（如滤镜、混合模式）可能无法完全还原
2. 某些特殊字体可能需要系统支持
3. 大文件解析可能需要较长时间
4. 浏览器内存限制可能影响大型PSD文件处理

### 🛠 故障排除

#### 服务器启动失败
```bash
# 检查端口占用
lsof -i :3000

# 强制重新安装依赖
rm -rf node_modules package-lock.json
npm install
```

#### PSD解析失败
- 确保PSD文件格式正确
- 检查文件大小（建议<100MB）
- 查看浏览器控制台错误信息

#### 文字显示异常
- 检查系统是否安装了相应字体
- 尝试使用通用字体（Arial、sans-serif等）
