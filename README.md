# PSD to Fabric.js 转换器

一个基于 Vue 3 的前端应用，可以将 PSD 文件转换为 Fabric.js 可识别的 JSON 格式，并提供画布预览功能。

## 功能特性

- 📁 **PSD 文件上传**: 支持拖拽和点击上传 PSD 文件
- 🔄 **自动转换**: 将 PSD 图层自动转换为 Fabric.js 对象
- 👁️ **实时预览**: 在 Fabric.js 画布上预览转换结果
- 🎛️ **交互控制**: 缩放、网格显示、图层管理等功能
- 💾 **JSON 导出**: 导出 Fabric.js 兼容的 JSON 文件
- 📱 **响应式设计**: 适配桌面和移动设备

## 技术栈

- **前端框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **PSD 解析**: ag-psd
- **画布渲染**: Fabric.js 5.x
- **文件处理**: file-saver

## 使用说明

### 1. 上传 PSD 文件

- 点击上传区域或直接拖拽 PSD 文件到指定区域
- 支持的文件格式：`.psd`
- 上传后会显示 PSD 文件的基本信息（尺寸、图层数量等）

### 2. 查看转换结果

转换完成后，会显示：
- **Fabric.js 画布预览**: 实时渲染转换后的内容
- **画布信息**: 尺寸、对象数量、当前缩放比例
- **图层列表**: 显示所有转换的图层，支持选择和隐藏/显示

### 3. 画布操作

- **适应画布**: 自动调整缩放以适应容器大小
- **重置缩放**: 恢复 100% 缩放比例
- **显示/隐藏网格**: 切换背景网格显示
- **鼠标滚轮缩放**: 在画布上滚动鼠标滚轮进行缩放
- **对象选择**: 点击画布上的对象或图层列表进行选择

### 4. 导出 JSON

点击 "导出JSON" 按钮可以下载转换后的 Fabric.js JSON 文件，该文件可以在其他 Fabric.js 应用中使用。

## 转换原理

1. **PSD 解析**: 使用 `ag-psd` 库解析 PSD 文件结构和图层数据
2. **图层处理**: 遍历 PSD 图层树，提取每个图层的画布数据
3. **图像转换**: 将图层画布转换为 base64 编码的 PNG 图像
4. **Fabric 对象创建**: 为每个图层创建对应的 Fabric.js 图像对象
5. **JSON 生成**: 组装成 Fabric.js 标准的 JSON 格式

## 支持的图层类型

当前版本主要支持：
- ✅ **图像图层**: 转换为 Fabric.js Image 对象
- ✅ **图层组**: 保持层级关系进行转换
- ✅ **图层属性**: 位置、尺寸、透明度、可见性等

## 注意事项

- PSD 文件大小建议控制在 50MB 以内，过大的文件可能导致处理缓慢
- 复杂的图层效果（如滤镜、混合模式等）可能无法完全保留
- 文本图层会被转换为图像，无法编辑文本内容
- 建议在上传前先在 Photoshop 中合并不必要的图层以提高转换效率

## 开发指南

### 添加新的图层类型支持

1. 在 `PsdConverter.vue` 的 `processLayer` 函数中添加处理逻辑
2. 根据图层类型创建对应的 Fabric.js 对象
3. 在 `FabricPreview.vue` 中添加相应的显示逻辑

### 自定义画布交互

可以在 `FabricPreview.vue` 中修改 Fabric.js 画布的事件处理和交互逻辑。
