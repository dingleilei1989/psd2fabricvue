# PSD 海报编辑器

一个基于 Vue 3 + ag-psd + Fabric.js 的在线海报编辑器，支持导入 PSD 文件并进行文字编辑。

## 功能特性

- 🎨 **PSD 文件导入**: 使用 ag-psd 库解析 PSD 文件
- ✏️ **文字编辑**: 添加和编辑文字，支持字体、大小、颜色设置
- 🔷 **形状工具**: 添加矩形、圆形等基本形状
- 🖼️ **图层管理**: 显示和操作 PSD 图层
- 🔧 **对象操作**: 删除、层级调整、复制等功能
- ⌨️ **键盘快捷键**: 支持常用操作的快捷键
- 💾 **导出功能**: 将编辑后的海报导出为 PNG 图片
- 🎯 **示例海报**: 内置示例海报生成功能

## 技术栈

- **Vue 3**: 前端框架（Composition API）
- **Vite**: 现代化构建工具
- **ag-psd**: PSD 文件解析库
- **Fabric.js**: Canvas 图形编辑库

## 使用说明

### 🎨 PSD 文件导入
1. 点击左侧工具栏的"导入PSD文件"按钮
2. 选择本地的 .psd 文件
3. 系统会自动解析文件并显示PSD信息和图层列表
4. 点击图层的"添加到画布"按钮将图层添加到编辑区域

### ✏️ 文字编辑
1. 在"文字工具"区域输入文字内容
2. 调整字体大小、颜色和字体家族
3. 点击"添加文字"按钮将文字添加到画布
4. 可以直接在画布上拖拽和调整文字位置

### 🔷 形状工具
1. 使用"形状工具"添加矩形或圆形
2. 可以自定义形状的颜色
3. 添加后可在画布上调整大小和位置

### 🔧 对象操作
- **选择对象**: 直接点击画布上的对象
- **删除对象**: 选中对象后点击"删除选中对象"或按Delete键
- **复制对象**: 选中对象后按Ctrl+D或点击"复制选中对象"
- **层级调整**: 使用"置于顶层"和"置于底层"功能

### ⌨️ 键盘快捷键
- `Delete/Backspace`: 删除选中对象
- `Ctrl+D`: 复制选中对象
- `Ctrl+S`: 导出图片
- `Escape`: 取消选择

### 💾 导出功能
点击"导出图片"按钮将当前画布内容保存为 PNG 文件

### 🎯 示例功能
点击"创建示例海报"按钮可以快速创建一个包含文字和形状的示例海报

## 开发说明

### 组件结构
- **左侧工具栏**: 包含文件操作、文字工具、形状工具、对象操作等功能
- **右侧画布区域**: 使用 Fabric.js 进行图形操作
- **PSD信息组件**: 显示导入的PSD文件信息和错误提示

### 关键功能实现

#### PSD 解析
- 使用 `ag-psd` 的 `readPsd` 函数解析PSD文件
- 递归提取图层信息，支持嵌套图层
- 将Canvas元素转换为DataURL用于Fabric.js

#### 画布操作
- 基于 `fabric.Canvas` 实现编辑功能
- 支持文字、图片、形状等多种对象类型
- 实现对象的选择、拖拽、缩放、旋转等操作

#### 事件处理
- 画布事件监听（选择、移动等）
- 键盘快捷键支持
- 文件上传处理

## 注意事项

1. **PSD文件大小**: 建议使用小于50MB的PSD文件以获得更好的性能
2. **图层复杂度**: 过于复杂的图层可能影响解析速度
3. **浏览器性能**: Canvas操作对浏览器性能有一定要求
4. **文件格式**: 仅支持标准的PSD文件格式
